- [x] **偏差分析副座標**：趨勢圖新增右側 Y 軸，動態顯示數據對比目標值的**偏離百分比 (%)**。
- [x] **圖表標籤增強**：為趨勢圖內的數據點、目標值、規格線及管制界限（UCL/LCL）新增數值標籤與標註。

- [x] **標準差多維分析**：計算並顯示組內 (Within)、組間 (Between) 與總體 (Overall) 變異指標。

- [x] **文檔庫 Wiki 化 (MECE)**：重整目錄結構，將歷史 `docs/references/` 升級為 Markdown 並歸檔至 `wiki/` 目錄。
